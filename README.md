
# BLACK-XP-BOT - شرح البوت

📌 تعريف:
هذا البوت هو بوت نقاط (XP) مخصص لزيادة التفاعل داخل سيرفر Discord.
يقوم بإعطاء المستخدمين نقاط عند النشاط مثل إرسال الرسائل.

---

⚙️ فكرة عمل البوت:

* كل مستخدم يحصل على XP عند التفاعل
* يتم تخزين البيانات في قاعدة MongoDB
* يتم تحويل XP إلى مستويات (Levels)
* يوجد نظام ترتيب (Leaderboard)

---

📂 الملفات الأساسية:

* example.js → الملف الرئيسي لتشغيل البوت
* package.json → يحتوي على المكتبات
* .env → يحتوي على معلومات حساسة (لا يتم رفعه)

---

🔐 شرح ملف .env:

BOT_TOKEN=YOUR_BOT_TOKEN
→ التوكن الخاص بالبوت من Discord Developer Portal

CLIENT_ID=YOUR_CLIENT_ID
→ معرف البوت (Client ID)

MONGO_URI=YOUR_MONGODB_URI
→ رابط الاتصال بقاعدة البيانات MongoDB

DB_NAME=discord-leveling
→ اسم قاعدة البيانات المستخدمة

---

🚀 طريقة التشغيل:

1. إنشاء ملف .env ووضع القيم الصحيحة

2. تثبيت المكتبات:
   npm install

3. تشغيل البوت:
   node example.js

---

📜 أوامر البوت (مثال):
/xp       → عرض نقاطك
/level    → عرض مستواك
/top      → عرض الترتيب
/help     → المساعدة

---

⚠️ ملاحظات مهمة:

* لا تشارك ملف .env نهائيًا
* تأكد من إضافة .env إلى .gitignore
* قم بتغيير التوكن إذا تم تسريبه

# .env 

BOT_TOKEN=
CLIENT_ID=
MONGO_URI=
